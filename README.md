# 🍴FruitFork: My Handwritten Font

![uzSCDZK](https://i.imgur.com/uzSCDZK.png)

我的手写字体——“水果叉子”，包含简体中文和英文两种字符。My handwritten font, called "FruitFork", contains Simplified Chinese and English characters.
